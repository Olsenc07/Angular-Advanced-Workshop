Exercise 1

Your mission is to turn the following dropdown into a country-selection dropdown.

  1. Open a terminal in your project ng-advanced-workshop
  2. Run npm run server to start the sample back-end that holds the data needed for all exercises
  3. Update the code in app/exercise1 so that our component loads its list of countries from http://localhost:3000/countries
  4. Display the total number of countries in the corresponding div
  5. Selecting a country should result in the id of the country being set as the dropdown value

You can take a look at the expected result here: http://localhost:4200/solution/1
