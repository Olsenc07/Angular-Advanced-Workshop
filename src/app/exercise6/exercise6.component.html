Exercise 6

Your mission is to create a solution that turns a clickable element into an action trigger with 3 states

  1. For instance, I could use that solution on a button with 3 possible text labels: Save, Saving..., and Saved!
  2. An action would be passed as an Observable. When the button is clicked, the action is triggered and the button text changes to Saving...
  3. Once the Observable returns a value, the button text changes to Saved!
  4. Try to make a reusable solution where everything can be customized, from the action to the text for the 3 states
  5. Your solution should work on any clickable HTML element, not just buttons

You can take a look at the expected result here: http://localhost:4200/solution/6
